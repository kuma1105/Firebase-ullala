# Firebase-ullala

Youtube 코딩애플

Firebase로 당근마켓 만들기 (맛보기 5강)

프로젝트 이름: ullala

초기 설치 명령어
npm install -g firebase-tools@9.12.1  

설치 후 로그인과 초기 설정
firebase login
firebase init

서버 가동
firebase serve -p 5001 -o 127.0.0.1
